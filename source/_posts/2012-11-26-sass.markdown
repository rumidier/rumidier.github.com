#### Passing Content Blocks to a Mixin

example.scss

    @mixin apply-to-ie6-only {
      * html {
        @content;
      }
    }
    @include apply-to-ie6-only {
      #logo {

      }
    }

    * html #logo {

#### Function Directives

example.scss

    $grid-width: 40px;
    $gutter-width: 10px;
    
    @function grid-width($n) {
      @return $n * $grid-width + ($n - 1) * $gutter-width;
    }
    
    #sidebar { width: grid-width(5); }
    
example.css

    #sidebar {
      width: 240px; }
