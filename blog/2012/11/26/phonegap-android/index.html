Phonegap-Android

Nov 26th, 2012 | Comments

사용 환경

  • 모델명 : MacBook Pro
  • 모델 식별자: MacBookPro8,1
  • 메모리: 4 GB
  • Eclipse: v4.2.1

요구 환경

  • Eclipse 3.4+

Eclipse Classic 설치

  • Eclipse Classic를 다운 받습니다.

  • 다운 받은 파일은 .tar.gz 압축 파일 이므로 압축 해제후 Eclipse 아이콘을 클릭 합니다.

  • 실행을 이어 나가면 다음과 같은 페이지가 나오며 ‘ok’를 클릭 합니다.

  • 설치 완료

Android SDK 설치

  • SDK에서 다운로드 가능 합니다.

Android SDK 환경 설정

파일 열기

$ vi ~/.bash_profile

.bash_profile 자신이 다운 받은 SDK파일의 경로를 추가해 줍니다.

설정 확인

$ source ~/.bash_profile
$ echo $PATH
…./Users/han/Development/android-sdk-macosx/sdk/platform-tools:/Users/han/Development/android-sdk-macosx/tools….

Xcode Command Line Tools 설치

Xcode 실행후 > Xcode Preferences 선택 합니다.

Downloads->Components-> Command Line Tools -> Install 클릭으로 설치 합니다.

설치가 끝나면 Terminal.app (iTerm)에서 커맨드 라인으로 작업을 할수가 있습니다. 자세한 사항은 Command-Line에서 iOS 부분을 참고 하시면 됩니다.

Apache Cordova 다운로드

다운로드된 파일은 .zip 파일이므로 압축을 풀어서 사용하시면 됩니다.

  • Cordova는 Cordova-Down에서 다운 받을수 있습니다.
  • 다운 받은 폴더를 ~/CordovaLib-2.x.x 쓰기 편한 위치로 이동 시킵니다.

새로운 프로젝트 시작하기

bin 폴더 열기

새로운 프로젝트를 생성하기 위해서는 ./create 파일을 실행 해야 합니다. create 파일은 lib/android/bin 폴더에 존재 합니다.

  • lib/ios/bin 폴더로 직접 이동 해서 사용할수 있습니다.
  • ~/CordovalLib-2.2/lib/android/bin 폴더를 Terminal.app에 드래그 하면 bin 폴더로 바로 이동 할수 있습니다.
create 실행하기
$ ./create <project_folder_path> <package_name> <project_name>

$ ./create <project_folder_path> pheonegap.start start
$ cd ~/project_folder_path
$ ls

AndroidManifest.xml cordova   project.properties
ant.properties                gen           res
assets                        libs          src
bin                           local.properties
build.xml                     proguard-project.txt

Simulator 실행하기

SDK 지정 하기

eclipse를 처음 실행 하게 되면 다음과 같이 SDK 폴더를 지정합니다.

SDK 경로 수정은 ‘Command + ,’ 사용해서 열수 있습니다.

Android SDK 매니저 설치

Android SDK 매니저 버튼을 클릭하거나 ‘Window -> Android SDK Mannager’를 클릭 합니다.

SDK 패키지 선택

버전에 맞는 API 선택하고 install package 버튼을 클릭 합니다.

SDK 패키지 설치

체크된 패키지를 설치 합니다.

Android ADK Plugin 설치

ADK Plugin에서 다운 받고 설치 가이드를 확인 할수 있습니다.

ADK 설치

Help -> insall new software를 클릭 합니다.

화면 오른쪽 위에 ‘add’ 클릭

Add Repository 설정

Location에서 문제가 발생하면 ‘https’를 ‘http’로 사용하도록 합니다.

Name: ADT Plugin
Location: https://dl-ssl.google.com/android/eclipse/

Available Software

Developer Tools과 필요한 프로그램을 클릭 한뒤 Next를 클릭합니다.

Install Details

인스톨 되는 정보를 확인후 ‘Next’를 클릭 합니다.

Review Lienses

라이센스 확인후 ‘Finish’를 클릭 합니다.

Security Warning

경고 화면이 나오게 되면 ‘Ok’를 클릭 하도록 합니다.

Software Update

설치가 끝나면 ‘Yes’를 클릭하여 새로 eclipse를 시작 하도록 합니다.

ADV(Android Virtual Device Manager)

ADV 버튼을 클릭하거나 ‘Window -> Anroid Virtual Devie Manger’를 클릭 합니다.

ADV New

오른쪽 상단에 ‘New’ 버튼을 클릭 합니다.

ADV Create

ADV 이름과 Device를 선택후 ‘ok’를 누릅니다.

Eclipse 실행

  • Command line tools를 사용 하여 새로운 프로젝트를 만듭니다.
  • 다운 받은 eclipse폴더에서 eclipse를 실행 합니다.
  • eclipse에서 ‘File->New->Project’를 클릭 합니다.
  • New Project 창에서 ‘Android Project from Existing Code’선택후 ‘Next’를 클릭 합니다.
  • 우측 상단의 ‘Browse..’ 클릭후 생성된 Cordova 프로젝트폴더를 엽니다.
  • Root Directory 정의후 ‘Run’실행 후 ADV를 선택 해주면 Android화면이생성
  • Phonegap Android는 컴파일 시간이 오래 걸리므로 기다려 주어야 합니다.
